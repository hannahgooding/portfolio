The Typic Witches

The website originally served 21 chapters of a novel. The client needed their server fixing as the domain was getting a DNS error. The client wanted to remove the existing chapters and replace them with 4 new chapters. The client wanted the chapters landing page link removed from the website, and the new chapters to be routed on a secret path so only users with the direct link can have access. Deployed on Netlify at www.thetypicwitches.com.
